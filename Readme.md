TechBlog – Django Blog Website

Projektbeschreibung:

Dieses Projekt ist eine Blog‑Webseite über verschiedene Programmiersprachen und Technologien. Die Webseite wurde mit dem Framework Django entwickelt. Benutzer können Blogartikel lesen, Beiträge liken und Kommentare schreiben. Ziel des Projekts war es, eine dynamische Blog‑Webseite mit verschiedenen Benutzerfunktionen zu erstellen.

Funktionen:

Anzeige von Blogartikeln über verschiedene Technologien
Filter nach Technologie (HTML, CSS, Python, Django, Java usw.)
Benutzer Registrierung und Login
Logout Funktion
Kommentare unter Artikeln schreiben
Kommentare löschen
Like‑Funktion für Beiträge
Passwort vergessen / Passwort zurücksetzen
Eigene 404 Fehlerseite
Technologien:

Python

Django

HTML

CSS

SQLite

Projektziel:

Mit diesem Projekt wollte ich lernen, wie man mit Django eine Blog‑Webseite mit Benutzerfunktionen, Kommentaren und Likes entwickelt.

Projektstatus:

Dieses Projekt ist ein Lernprojekt und Teil meines Portfolios im Bereich Webentwicklung.
